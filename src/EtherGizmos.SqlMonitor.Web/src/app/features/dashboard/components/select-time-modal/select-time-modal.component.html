Relative dates start with a t, representing the current time (now), and may either add (+) or remove (-) any number of date units. Valid date units are year (Y), month (M), week (w), day (d), hour (h), minute (m), and second (s).
Additionally, the date can be rounded up or down to a specific unit by specifying either the start (@soX) or the end (@eoX) of one of the above units, X.
